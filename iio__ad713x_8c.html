ssize_t iio_ad713x_read_dev ( void *  iio_inst,
char *  pbuf,
size_t  offset,
size_t  bytes_count,
uint32_t  ch_mask 
)

Read chunk of data from RAM to pbuf. Call "iio_ad713x_transfer_dev_to_mem" first. This function is probably called multiple times by libtinyiiod after a "iio_ad713x_transfer_dev_to_mem" call, since we can only read "bytes_count" bytes at a time.

Parameters
iio_inst- Physical instance of a device.
pbuf- Buffer where value is stored.
offset- Offset to the remaining data after reading n chunks.
bytes_count- Number of bytes to read.
ch_mask- Opened channels mask.
Returns
bytes_count or negative value in case of error.
